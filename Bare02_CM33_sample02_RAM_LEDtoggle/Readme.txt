This project is a RAM based project.
	There is no use of FLASH memory. If you switch off,  the program is gone.

Application: LED-ON/OFF project(without Semihosting)
 This project can be worked with EWARM.
 
EWARM: 9.60.3(with RP2350 patch)
Debugger otpion: --drv_vector_table_base=0x20000000
SRAM: 0x20000000-0x20081FFF
